Color Variables
You can use a palette of 32 built-in colors. Each color is manually generated without auto-generation. For each color, you can use three different variants:
  • light --color-light-*
  • minor --color-minor-*
  • main --color-*
  • saturated --color-sat-*
  • dark --color-dark-*

.box {
    background: var(--color-light-red);
}
Color Palette
Accents

You have a set of colors to emphasize the user's attention: .primary, .secondary, .success, .warning, .alert, and .dark. These classes also add a color change when hovering over an element and set the white color for the text.

Background Colors

Metro UI provides the user with various classes to set the background element.

General Background

Use classes bg-*** to set the background color for the element. These classes change only the background color.

Opacity

Use classes op-***, .op-***-hi, and .op-***-low to set the background color with opacity for the element. The alpha value is: .op-*** - 50%, .op-***-hi - 20%, and .op-***-low - 80%. These classes change only the background color.

Gradients

Use classes gr-*** to set the background color with a gradient. These classes change only the background color. You can change gradient direction with css var --gradient-direction

Ribbed

Use classes ribbed-*** to create a ribbed background. These classes change only the background color. You can use CSS variables --ribbed-size, --ribbed-alpha, and --ribbed-angle to change visualizing.

Soft & Normal

You can use classes soft-*** and normal-*** to change the background, foreground, border and hover colors. These classes usefulness for buttons.

Borders

You can change border and outline colors with classes bd-*** and ol-***.

Border Color

To set a border color use classes bd-***. A special classes .animated-border, .animated-border-*** will add an animated border to your element. To use animated border, the element must have a background and a parent element must have a z-index.

Outline Color

To set color for element's outline use classes ol-***.

Fill & Stroke

You can use classes fc-*** and st-*** to change the fill and stroke color properties for svg elements.

Pseudo Elements

With classes .before-bg-***, .before-fg-***, .after-fg-***, and .after-fg-*** you can change colors for pseudo-elements.
