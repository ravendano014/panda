What is Common CSS?
Common CSS is a collection of utility classes that are part of the Metro UI framework. These classes allow you to:
  • Apply common styling patterns directly in your HTML
  • Create responsive layouts with minimal custom CSS
  • Maintain consistent spacing, typography, and colors across your application
  • Quickly prototype and build interfaces

Metro UI contains a lot of common CSS classes that you can use in your projects. These are classes for flexbox, alignment, shadows, and more. And yourself:

Border

Use classes .border, .border-top, .border-right, .border-bottom, .border-left to set border. With classes .border-solid, .border-dashed, .border-dotted, .border-double, .border-groove, .border-inset, .border-outset, and .border-ridge, you can set border style.

With classes .border-size-1....border-size-20, you can set border size. With classes .border-radius-1....border-size-20, you can set border radius.

To change border color, use classes .bd-color_name.

With classes .animated-border, .animated-border-hover, and .animated-border-[color] you can add special border to elements.

Display

Use classes .d-block, .d-inline, .d-inline-block, .d-flex, .d-grid, .d-table, .d-table-cell, .d-none, .visible, .no-visible to set display property for an element.

Embed Containers

Use class .embed-container to embed a video or other content. You can set an aspect ratio using additional classes .size-21x9, .size-16x9, .size-4x3, and .size-1x1.

FlexBox

You can use built-in classes to create a flexbox layout. With these classes, you can set flexbox container and flexbox items. All classes you can find in the Metro UI Flexbox

Images

You can create image containers with different sizes and fit options. To create an image container, use classes .img-container, .img-fluid .img-thumbnail. You can add an overlay to the image using the .img-overlay class.

Positioning

Use classes .pos-relative, .pos-absolute, .pos-fixed, .pos-sticky, .pos-static, .fixed-top, .fixed-bottom to set position property for an element.

Spacing & Sizing
The spacing module defines CSS classes for adding or removing space around elements using margin and padding properties. These utilities follow a consistent naming convention and scale, making it easy to apply spacing in a standardized way throughout your application.
The sizing module defines CSS classes for setting element dimensions using various units (percentages, viewport units) and constraints (min, max). These utilities make it easy to create responsive layouts with precisely sized elements.
