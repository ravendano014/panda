Media Rules

Metro UI is designed as a mobile-first system. We use min-width property for building media queries.

Breakpoints:
BREAKPOINT RULE SIZE PX
fs min-width 0
sx min-width 360
sm min-width 576
ld min-width 640
md min-width 768
lg min-width 992
xl min-width 1200
xxl min-width 1400
xxxl min-width 2000
Using Media Rules

1 You can use media rules to create responsive layouts with a grid system. To use media rules, add class .cell-BP-SIZE, .col-BP-SIZE, or .cell-BP-SIZE (for query container) to the element. Where BP - breakpoint, and SIZE - size of the cell.

2 You can use media rules in components with attribute data-expand-point="*".
Where * - breakpoint. More details in the documentation for the components on the official website.

3 With utility function Metro.utils.mediaExist(BP) and Metro.utils.inMedia(BP).
Where BP - is breakpoint..
