Dark & Light
New

Metro UI supports out-of-the-box light and dark themes for components. By default, the theme is set to light. To change the theme, add class .dark-side to the <html> element for global or set this class to the <element> for local.

Theme according to the system

You can use the prefers-color-scheme media feature to detect if the user has requested the system use a light or dark color theme. To use this feature, add meta tag metroui:theme with content auto to the page head. Also, you can use values light and dark to set the specific theme.

The component theme-switcher will not work if this meta-tag is used!
Theme Switcher

You can use component theme-switcher to add control to switch between light and dark themes. This component automatically sets theme class .dark-side to the specified in attribute data-target element (by default it is <html> element). To use this component, add attribute data-role="theme-switcher" to the <input> element.
