About Touch and Swipe

Metro UI supports touch and swipe events. You can use the data-role="swipe" attribute to enable swipe events for the element. You can use events data-on-swipe, data-on-swipe-left, data-on-swipe-right, data-on-swipe-top, and data-on-swipe-bottom to handle swipe events.

Swipe on red box to see the swipe log.
