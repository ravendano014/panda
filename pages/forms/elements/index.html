About Form Elements

Panda uses a more powerful input component provided by the Metro UI library. This component turns a regular input into a multipurpose user input tool. You can use text inputs, checkboxes, radio buttons, text area, and other input types.

Inputs

Use standard HTML input.

Extended Inputs

Use standard HTML input with attribute data-role="input" to create input control with extended features.

You can use built-in search button.
You can use built-in randomizer button.
Use Enter to store value, and Up, Down to access to history
Checkboxes and Radio Buttons

Use standard HTML input to create stylized controls, add class checkbox-*** or radio-*** (where *** is color name) to change control color.

Checkboxes
Extended Checkboxes

Use attribute data-role="checkbox" to create stylized controls with extended options.

Custom Checkboxes

Use attribute data-role="custom-checkbox" to create stylized controls with extended options.
