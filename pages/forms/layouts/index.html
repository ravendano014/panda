Form Elements Containers

You can use next container classes for elements: .form-group and .form-actions.

Default Layout

In the default layout, form elements are placed in columns.

Inline Layout

In the inline layout, form elements are placed in a row. To use inline layout, add class .inline-form to the <form> element.

Grid Layout

In the inline layout, form elements are placed in a grid. To use grid layout, add class .grid-form to the <form> element. To change gap size, set CSS variable --form-gap-size to required value. Min column size in the form grid - 200px, to change this value, set CSS variable --form-column-min-width to required value.
