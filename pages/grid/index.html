About Grid System

You can create a grid with built-in grid system. The grid system uses a series of containers, rows, and columns to layout and align content. It is built with CSS container query, flexbox and is fully responsive.

The following CSS classes are available for building a grid: .grid, .row, .cell-*, .cell-*-*, .cell-*-one-fifth, .cell-*-two-fifth, .cell-*-three-fifth, .cell-*-four-fifth, .offset-*, and .offset-*-*.

Grid uses @container queries and adapts depending on the size of the closest parent with the .row class.
