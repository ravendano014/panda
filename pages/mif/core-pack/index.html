About Icons

Panda 1.0 uses Metro UI Icons. Metro Icons is an open source SVG icon library featuring over 800+ glyphs and distributed as a font file. You can include icons in your project either using a CDN or using your favorite bundler.

To use icons, you need to add class `mif-icon_name` to the HTML element.

To change color, use classes fg-*** (where *** is color name).

You cna use any icon size by adding mif-Nx class (where N is a number from 1 to 10).
