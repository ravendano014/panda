Notify

You can create notify message for user with method Metro.notify.create(message, title, {options}) 


            Metro.notify.create('Notify message', 'Title')

You can change notify visibility time with timeout option. To change notify style you can use clsNotify option or set CSS vars: --notify-background, --notify-color, --notify-border-color.

You can use callback functions: onNotifyCreate, onClick, onClose, onShow, and onAppend to handle notify events.
Toast

You can create toast message for user with method Metro.toast.create(message, {options})

You can change notify visibility time with timeout option. To change notify style you can use clsToast option or set CSS vars: --toast-background, --toast-color. Also, you can change toast position with data-position="..." option (available values: top, bottom, and center, ).

You can use callback function: callback to handle toast close event.
