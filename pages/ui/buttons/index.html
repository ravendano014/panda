Buttons

  • Panda 1.0
  • UI
  • Buttons
Default Buttons

Use the button classes on an <a>, <button>, <input>, or <span> element.

Rounded Buttons

Add .pill-button to a button to make it rounded.

Shadowed Buttons

Add .shadow to a button to make it shadowed.

Shifted Buttons

Add .shift to a button to make it shifted.

Outline Buttons

Use a class .outline to create outline buttons.

Outline Rounded Buttons

Add .pill-button to a button to make it rounded.

Soft Buttons
New

Use a class .button-soft-*** to create soft buttons where *** is the color name.

Colored Buttons
New

Use a class .button-*** to create soft buttons where *** is the color name.

Icon Buttons

You can use any icon fonts or image to add icon into button. You can create these buttons with or without a caption.

Button Sizes

Add classes large, small, or mini to change button size.

Button Types

Add classes .square, or .cycle to change a button type.

Button with Animated Border
New

You can use classes .rainbow-border, .rainbow-border-hover, .animated-border, .animated-border-hover, and .animated-border-*** (where *** is the color name) to create buttons with animated border.
