Use class .badge to create badges. Use accent classes, .soft-** or .normal-** color classes to create colorized badges.

Tag

Tag is used by the select and tag-input components, but you can also use the tag element classes at your discretion. Use accent classes, .soft-** or .normal-** color classes to create colorized badges. Unlike chips, it doesn't contain an icon.
