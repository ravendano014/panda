Shortcut

Create a shortcut button with .shortcut class.

You can use soft-* and normal-* classes to change shortcut color.

Toolbar & Tool Button

You can use .toolbar and .tool-button classes to create toolbar.

Use class .outline for button to create outline button.

Add attribute data-caption to add caption to toolbar.

Transform Button

Use class .transform-button to create transform button. For you available transform classes: .up, .down, .left, .right, .top-left, .bottom-left, .top-right, and .bottom-right.

Click the buttons below to view transform action.

Button Group

Create a button group with Button Group component. You can create a group with radio state or multi state with attribute data-mode="multi".
