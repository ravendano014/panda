About

You can create split blocks with component splitter. You can split block horizontally or vertically. To create a splitter, you need to add attribute data-role="splitter" to the element and add child blocks.

Panels Sizes

You can setup initial sizes for split panels with attribute data-sizes="x, y". The size of the first panel is x and the size of the second panel is y. You can set sizes in percents or pixels. For example, data-sizes="30%, 70%" or data-sizes="220," (set only first N elements size, and omit the rest). To save state in storage, you need to add id to splitter and attribute data-save-state="true".

Gutter Size and Type

You can setup gutter size and type with attribute data-gutter-size="x". The size of the gutter can be in pixels. To change the gutter type, you need add attribute data-gutter-style="x". The type of the gutter can be ribbed, dashed or dotted.
