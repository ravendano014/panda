Table Component
You can transform table to table component with attribute data-role="table". The Table component is a powerful and flexible data grid that allows you to display, sort, filter, and manipulate tabular data. It supports various features like pagination, searching, row selection, custom cell rendering, and more.
Table component has many options and opportunities вЂ” for example, you can redraw table cell with event onDrawCell. Detailed description for all options available in the documentation for the Table Component.
