About Tiles

Metro UI contains classes to create tiles such as application tiles in Windows 8/10. To create a tile, you can add role tile to an element with attribute data-role.

Tiles Grid

For position tiles, you must use a special container with class .tiles-grid. The grid is divided into cells of size 70x70 with an interval of 10px without limitation to height or width. You can limit the size of the grid with subclasses .size-* from 1 to 10. One size is equivalent to a small tile. All sizes include a gap.

Position in a grid

You can place a tile to a specified position in grid with special classes .col-* and .row-*. These classes defined from 1 to 12 and allow you to place the tile in the specified column and row.

Grouping tiles

To grouping tiles, you can use special class .tiles-group. To set group fixed size, add one of classes: .size-half, .size-1, .size-2, .size-3, .size-4, .size-5, .size-6, .size-7, .size-8, .size-9, .size-10.

Also, you can use media breakpoints for change size: .size-{media}-half, .size-{media}-1, .size-{media}-2, .size-{media}-3, .size-{media}-4, .size-{media}-5, .size-{media}-6, .size-{media}-7, .size-{media}-8, .size-{media}-9, .size-{media}-10. Where {media} is a breakpoint: fs, xs, sm, ld, md, lg, xl, xxl, and xxxl.

Tile Size

You can use 5 sizes for your tiles: app, small, medium, wide and large. To set tile size use attribute data-size. The default size is medium.

Tile Content
Icon

To add icon to tile, you can use .icon class. You can use image or icon class.

Branding Bar

The branding bar is a text on the bottom left of the tile. To add a branding bar, you can use .branding-bar class.

The Rocket
Badge

You can add a badge to tile. Use classes .badge-top and .badge-bottom to add a badge.

Image

You can add cover image to tile. Use attribute data-cover to add an image.

Image Set

You can use live wide tile with image set effect. To create this tile, add attribute data-effect="image-set" and add images inside tile.

Tile Color

You can change the background color of tile with built-in background color classes or inline CSS.
