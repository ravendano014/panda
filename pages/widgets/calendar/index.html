About

Create calendar with role data-role="calendar".

Compact Mode

Use attribute data-compact="true" to set compact mode.

Week Start

The Calendar can draw week start from Sunday or Monday. This option sets automatically based on the current locale. For lang=en week starts from Sunday, for lang=ua week starts from Monday. Also, you can use attribute data-week-start="0" to set week start from Sunday or data-week-start="1" to set week start from Monday.

i18n

The Calendar can be localized to different languages. The default language is English. You can set the language using the lang attribute.

Show Time

Use attribute data-show-time="true" to show time.

Calendar Options

You have many options for displaying and selecting date(s) on the calendar. For more information, see the Metro UI documentation.
