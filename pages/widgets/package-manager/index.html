About

The package manager component is designed to visualize the installation of a package from npm. With Metro UI, you can create package manager simple and easy with role package-manager. The package manager is ideal for displaying the installation of a package from npm.

All these components interconnected. To disable or change link, use the data-link attribute.

Default tab

You can change a default tab with attribute data-default and value: npm, pnpm, yarn, or bun.
