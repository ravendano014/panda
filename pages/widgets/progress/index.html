Progress Bars

Use progress component to create progress bars. To set value use attribute data-value.

To create thin progress bar, add attribute data-small="true".

To change bar color use color classes bg-***, soft-***, or normal-*** and attribute data-cls-bar. Or use classes .progress-*** where *** is color name or accent name.

Use donut component to create donut progress bars. To set value use attribute data-value. To change color, use classes .donut-*** where *** is color name or accent name.
